## ✨ About MegaMart

<p align="center">
MegaMart is a <b>premium E-commerce landing page</b> inspired by real-world platforms like Flipkart & Amazon.<br/>
Designed with modern UI principles using <b>React.js + Tailwind CSS</b>.
</p>
